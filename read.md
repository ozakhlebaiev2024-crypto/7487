2. Математичні та Логічні Оператори
Використовуйте оператори для обчислень (+, -, *, /) та порівнянь (>, ==).

Приклад коду:

Python

a = 10
b = 3

# Математичний оператор (остача від ділення)
remainder = a % b
print(f"Остача: {remainder}")

# Логічний оператор
comparison = a > 15
print(f"a > 15: {comparison}")
Результат виконання:

Остача: 1
a > 15: False
3. Структури Даних
Списки (Lists)
Змінна (mutable), впорядкована колекція.

Приклад коду:

Python

my_list = ["A", "B", "C"]
my_list.append("D")
print(f"Список: {my_list}")
print(f"Елемент з індексом 0: {my_list[0]}")
Результат виконання:

Список: ['A', 'B', 'C', 'D']
Елемент з індексом 0: A
Словники (Dictionaries)
Невпорядкована колекція пар ключ-значення.

Приклад коду:

Python

config = {"user": "guest", "timeout": 300}
print(f"Значення timeout: {config['timeout']}")
config['timeout'] = 600
print(f"Нове значення timeout: {config['timeout']}")
Результат виконання:

Значення timeout: 300
Нове значення timeout: 600

Python

else:
    print("Мала")

Середня
Цикл for
Для перебору елементів колекції або послідовності.

Приклад коду:

Python

for num in range(3): # range(3) дає 0, 1, 2
    print(f"Квадрат {num}: {num ** 2}")
Результат виконання:

Квадрат 0: 0
Квадрат 1: 1
Квадрат 2: 4
